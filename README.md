# BLIP Image Captioning and VQA

Image captioning and Visual Question Answering (VQA) using BLIP (Bootstrapping Language-Image Pre-training) from Hugging Face.

## Overview

This project demonstrates two key Vision-Language tasks:
1. **Image Captioning**: Generate natural language descriptions for images
2. **Visual Question Answering**: Answer questions about image content

## Model

- **BLIP-base** (Salesforce/blip-image-captioning-base) for captioning
- **BLIP-large** (Salesforce/blip-image-captioning-large) for VQA

## Features

- Load images from local or URL
- Generate descriptive captions
- Answer questions about images
- Support for multiple BLIP model variants

## Technologies

- Hugging Face Transformers
- PyTorch
- PIL (Python Imaging Library)
- BLIP (Bootstrapping Language-Image Pre-training)
